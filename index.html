Komandų aprašymas

    Sąrašo komandos

    Į pradžią
  • ls (parodo visus failus ir katalogus esamoje direktorijoje)
  • ls ~ (parodo visus failus ir katalogus esančius Home kataloge)

    • Naršymo po katalogus komandos

    Į pradžią
  • cd foldername (įeiname į katalogą foldername (gali būti kitas pavadinimas arba visas keles iki reikemos vietos) esamoje direktorijoje jei tokiu pavadinimu turime jau katalogą). Taip pat skaitykite CD Papildoma naudinga info.
  • cd .. (einame vienu žingsniu aukštyn/atgal iš esamo katalogo)
  • cd ../.. (du žingsniai atgal (cd ../../.. bus trys žingsniai))
  • cd ~ (grįžta į home katalogą)
  • cd ..; (kabliataškis ; leidžia parašyti dvi ar daugiau komandas pvz.: cd ~; ls; cd desctop "sugrįš i Home katalogą iškvies Home katalogo dokumentų sąrašą ir nukeliaus į desktop direktoriją/aplanką"

    • Katalogo ar failo kūrimo komandos

    Į pradžią
  • mkdir foldername (sukurs naują katalogą esamoje direktorijoje galima ir kelę nurodyt kur sukurtų naują katalogą jei jau turim katalogą kuremia norime sukurti naują aplanką pvz.: mkdir foldername/naujaskatalogas "sukurs naują katalogą pavadinimu "naujaskatalogas" kataloge foldername")
  • touch index.html (sukurs nauja index.html failą esamame kataloge
  • Pvz.: tarkim esamme desktop kataloge ir į terminalą įrašom komandas: mkdir naujas01; mkdir naujas01/01; touch naujas01/01/index.html - tai reiškia, kad pirmas mkdir sukurs kataloga pavadinimu naujas01, antrasis mk dir po ; sukurs kataloge naujas01 katalogą pavadinimu 01 ir po ; komanda touch sukurs kataloge 01 failą pavadinimu index.html ,o kad sukurtų 01 aplanke index.html nurodom kelę iš esamo aplanko naujas01/01/

    • Direktorijų ir failų vietos manipulecijos komandos

    Į pradžią
  • pwd (terminale parodo kokioje vietoje esame visą kele iki esamo katalogo)
  • mv "file.txt" directorijos pavadinimas (perkels faila kurio pavadinimas file.txt (plėtinys .txt nebūtinai gali būti matomas) į mūsų pasirinktą direktoriją pvz.: jei esame desktop aplanke/direktorijoje ir joje turime failą pavadinimu file.txt ir iš ankstesnio pvz. turime aplanką kurio pavadinimas naujas01 tai terminalę įrašę komanda mv file.txt naujas01 perkelsime file.txt į aplanką naujas01)
  • mw *.jpg direktorijos pavadinimas leis perkelt visus failus į nurodytą direktoriją su plėtiniu jpg nepriklausomai nuo failo pavadinimo. Pvz.: *.txt perkeltų visus failus iš esamos direktorijos/aplanko į nurodytą aplanką/direktoriją
  • mv direktorijos_pavadinimas/index.html naujas_direktorijos_pavadinimas (leižia neišeinant iš esamos direktorijos/katalogo perkelti failus tarp kitų katalogų). Pvz.: mv naujas01/01/index.html naujas01 - naujas01/01 nurodo aplankus/kelę iki norimo perkelti failo kuris yra index.html tada nurodom kur norime perkelt, šiuo atveju į naujas01 aplanką.

    • Curl ir failų peržiūrėjimas

    Į pradžią
  • Curl reiškia C URL. Norėdami pamatyti tinklapį terminalo lange
    • culr -L http://www.google.com (-L nukreipai į naują adresą jei puslapio adresas pasikeites ir puslapio savininkai nurodę tą pasikeitimą).
    • culr -o google.html -L "http://www.google.com" (-o nurodo koks faulo pavadinimas ir plėtinys bus šiuo atveju html bet taip pat galima ir txt išsisaugot nurodytą puslapį).
        • Failų peržiūrėjimas (cat, less)
      • cat file.txt (leidžia peržiūrėti txt pletinio failą esamoje direktorijoje terminalo lange. Pvz.: cd Desktop/naujas01 cat file.txt (leistu būnant Home aplanke/direktorijoje nurodant kelę iki file.txt atidaryti jį terminalo lange).
      • less file.txt (leidžia redaguoti failus consolėje, spaudžiam v (kad pradėt editint) padarom edit kažkokį tuomet esc ir :wq! (išsaugo failą ir išeina)).

    Failų ir aplankų ištrynimo komandos

    Į pradžią
  • rm file.txt (ištrins faila pavadinimu file su galune txt iš esamos direktorijos)
  • rmdir foldername (ištrins direktor/aplanką neveiks jei direktorijoje bus kita direktorija ar failai)
  • rm -rf foldername (ištrins direktoriją ir turinį joje -rf neklaus ar tikrai norime ištrint (tas pats suveike tik su -r nenaudojant f)*)

    • Atverti darbinę direktoriją

  • open . (reiktų patikslinimo ką daro šita komanda)*
  • code . (atidarys Visual Studio Code su esamu aplanku). Pvz.: tarkim esam direktorijoje/aplanke Home tai įrašius komandą cd Desktop/naujas01/01/ ; code . VSC atidarys aplanką 01 esanty desktop aplanke naujas01.

    • Pagrindinės git komandos

    Į pradžią
  • echo "# test2">>README.md
  • git init (init comanda naudojama kai norima pradėti naują repositoriją arba inicijuoti esamą)
  • git add README.md
  • git add . (naudojama pakeistiems failams sukurti)
  • git commit -m "first commit" (lokalus išsisaugojimas momentinė versija)
  • git remote add origin git@github.com:yourUsername/test.git (prideda naują repozitoriją)
  • git push -u origin master (nusiunčia lokalius pakeitimus į tinklą)
  • git reset --hard (atstato failus į pradinę buseną/versiją)
  • git chechout -b branchName (skirta pereitį iš vienos šakos į kitą šaką)
  • git pull origin master (dirbank keliese nu viena šaką paimtu visus pakeitimus susijusius)
  • git stash
  • git diff - parodo lokalius pakeitimus
  • git clone "repository url" (skirta parsisiusti naujausę versiją ir nukopijuoti į pasirinktą vietą)

    Paruoštukas

    Į pradžią
  • 1-10 žingsniai reikalingi tik pirmą kartą sukūrus projektą.
  1. Nueiname į direktoriją kur kursime aplanką
  2. su "mkdir" sukuriame aplanką (be tarpų ir simbolių!)
  3. einame į naująjį aplanką
  4. kuriame failus su "touch" (index, style)
  5. sukuriame (inicijuojame) repozitoriją su "git init"
  6. pridedame pavienius failus su "git add" arba visus su "git add ."
  7. commitinam failus į repozitoriją su "git commit -m "komentaras kas atlikta""
  8. sukuriame git hube repozitoriją
  9. susiejame abi repozitorijas su: "git remote add origin"
  10. išstumiam failus į git hub su "git push origin master"
  • 1-4 jau po to darant tolimesnius pakeitimus.
  1. atliekame pakeitimus, geriausia kad jie būtų grupuojami pvz spalvų pakeitimas css
  2. pridedame pakeistus failus arba visus direktorijoje esančius failus "git add ."
  3. commitiname su komentaru kas atlikta
  4. išpušinam į github

    Papildoma naudinga informacija

    Į pradžią
  • Paspaudus shift+ins terminalo lange bus atliekama komanda įklyjuoti (paste)
  • Paspaudus shift+arrow key (shift klavišas + strėlytė į kairę arba dešinę) - žymeklis juda iki artimiausio simbolio (žodžio raidės nėra simbolis šiuo atveju)
  • Į terminalo langą įtempus failą arba aplanką parašoma pilna direktorija kur randas failas ar aplankas
  • Į terminalo langą įrašius komandą cd ir įtempus aplanką bus nueitą į aplanko direktoriją
